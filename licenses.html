AIUL License Tags

The AI Usage License (AIUL) framework includes six primary license tags, each representing a different level of permitted AI usage. These tags can be combined with optional media-specific suffixes to provide additional clarity for different types of assignments.

Primary License Tags

Tag Description Example Use Case
AIUL-1 No AI tools allowed. All work must be entirely student-generated. Skill-building exercises, traditional technique development, handwritten exams
AIUL-2 Limited AI assistance is permitted only with instructor pre-approval. Assignments where AI might be helpful for specific students with accommodations
AIUL-3 AI tools can be used for research or inspiration only. Final work must be original. Research papers, concept development, preparatory work
AIUL-4 AI is used as a collaborative tool. Significant student input is required. Iterative creative projects, draft development, AI-assisted writing
AIUL-5 AI-assisted creation permitted with clear student direction and modification. Projects focusing on concept and direction rather than technical execution
AIUL-6 This assignment requires integrated AI usage. The process and final work must reflect this integration. Courses focused on AI tools, AI ethics assignments, AI prompt engineering

Optional Media-Specific Suffixes

These suffixes can be added to any of the primary license tags to provide additional context for specific types of media:

Suffix Media Type Example Tools
-WR Writing ChatGPT, Claude, Bard, language models for text generation
-IM Images DALL-E, Midjourney, Stable Diffusion, image generators
-VD Video Runway, Synthesia, Descript, video generation/editing tools
-AU Audio Descript, Murf, ElevenLabs, audio generation tools
-3D 3D Models/Animation Blender AI tools, Plask, ML-assisted 3D modeling
-TR Traditional Art AI assistants in traditional art contexts
-MX Mixed Media Combination of multiple AI-assisted media types

Examples of Combined Tags

  • AIUL-1-WR: A writing assignment where no AI text generation is permitted.
  • AIUL-3-IM: An image creation assignment where AI can be used for inspiration or research but the final image must be created without AI generation.
  • AIUL-4-AU: An audio production assignment where AI tools can be used collaboratively, but with significant student input and direction.
  • AIUL-5-VD: A video project where AI-assisted creation is permitted with student direction and editing.
  • AIUL-6-MX: A mixed-media project that requires the integration of AI tools as a core learning objective.
