Welcome to the AI Usage License Project

The AI Usage License (AIUL) project provides a standardized framework for educational institutions to clearly communicate expectations around AI tool usage in academic assignments.

Our Mission

As AI tools become increasingly prevalent in education and creative fields, educators need clear ways to communicate their expectations for how and when students can use these tools in their coursework. Inspired by the Creative Commons project, the AIUL project creates a simple tagging system that can be applied to any assignment, making expectations transparent and consistent.

How It Works

Each AIUL tag indicates a specific level of permitted AI usage, from completely prohibited to fully integrated. Optional media-specific suffixes can be added to provide additional clarity for different types of assignments.
